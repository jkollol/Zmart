Zmart – Modern eCommerce Platform

Overview

Zmart is a full-stack eCommerce platform designed to deliver a smooth shopping experience for users and an easy product-management system for administrators.
This project was built to practice and implement real-world eCommerce features using Laravel, Bootstrap, MySQL, authentication systems, file validation, Blade components, middleware, and Eloquent relationships.

⸻

Features

1. Product Management (CRUD)
	•	Add, edit, update, and delete products
	•	Image upload with validation
	•	Product descriptions, pricing, discounts, quantity
	•	Organized display of products under categories

2. Category and Subcategory Management
	•	Full CRUD operations for categories
	•	Eloquent relationships between categories and products
	•	Easy browsing and filtering of products by category

3. User Authentication
	•	Registration and login system
	•	Middleware-based route protection
	•	Password reset option (if enabled)
	•	Secure session handling for users

4. Shopping Cart System
	•	Add products to cart
	•	Update item quantity
	•	Remove items
	•	Automatic total and subtotal calculation
	•	Persistent cart for logged-in users

5. Checkout Flow (Basic)
	•	Order summary
	•	Customer details
	•	Order placement
	•	Payment placeholder (future integration supported)

6. User Interface and Design
	•	Responsive layout using Bootstrap
	•	Reusable Blade components
	•	Clean design with product cards, navigation bar, and footer
	•	Proper use of alerts and notifications for user actions

7. Admin Dashboard
	•	Manage products, categories, and orders
	•	View customer list
	•	Role-based access control (Admin/User)
	•	Secure backend accessible only to administrators
